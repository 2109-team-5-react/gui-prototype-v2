# Usage
<!-- usage -->
```sh-session
$ npm install -g synapse
$ synapse COMMAND
running command...
$ synapse (-v|--version|version)
synapse/0.0.0 linux-x64 node-v16.5.0
$ synapse --help [COMMAND]
USAGE
  $ synapse COMMAND
...
```
